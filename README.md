Simple Interest Calculator
A simple Bash script that calculates simple interest given the principal amount, annual rate of interest, and the time period in years.

Purpose
This project is built as part of a GitHub and Git fundamentals project to practice repository management, version control workflows, branching, and automation scripts.

Usage
To run the script, open your terminal and execute:
bash simple interest sh

Input Fields
The script dynamically prompts the user to enter the following values:

Principal: The initial amount of money borrowed or invested.

Rate of Interest: The annual interest percentage rate.

Time Period: The duration of the loan or investment in years.

Formula Used
The simple interest is calculated using the standard formula.
